# часть 1
## сценарии использования
### Поиск товаров
1) Через меню снизу зайти на страницу "Товары"
2) Сверху в панели "фильтры" заполнить требуемые параметры (категория товара, поставщик, срок хранения, наименование и тд)
3) Нажать кнопку "поиск" рядом с панелью фильтров
4) Снизу выводится список строк товаров, нажатие на них сворачивает/разворачивает в три кнопки "Изменить"/"Удалить"/"Страница товара", отдельная страница товара выводит более подробную информацию о товаре, включая список соответствующих товарных единиц на складе с информацией о каждой из них.
5) На странице товара есть дополнительный фильтр для товарных единиц (поставщик, местоположение, объем).

### Поиск заказов/поставок за период времени
1) Перейти через меню на страницу "Заказы"/"Поставки"
2) В панели фильтров заполнить диапазон времени, поставщика/потребителя, наименование товара
3) Нажать кнопку "поиск" рядом с панелью фильтров
4) Результаты выводятся снизу панели в виде списка разворачиваемых записей. Нажатие на переносит на страницу заказа/поставки (выводит на ней более подробную информацию о поставке/заказе, в том числе ссылку на поставщика/потребителя, список товарных единиц).

### Оформление нового заказа/поставки
1) Перейти через меню на страницу "Заказы"/"Поставки"
2) Нажать кнопку "+" - перейти на страницу создания или изменения заказа/поставки
3) Заполнить дату, товар(по id или наименованию) потребителя/поставщика и т.д.
4) Нажать кнопку "сохранить"

### Свободное место для поставки
1) Место для поставки резервируется при ее создании и сохраняется в информации о товарных единицах с привязкой к данной поставке. При нехватке места поставка не оформится, ниже окна заполнения выведется информация о свободном месте и требуемом для поставляемых товаров месте. (В базе данных хранится значение загруженности каждой полки).
2) После успешного создания записи о поставке можно посмотреть зарезервированное под него место, нажав на поставку и перейдя на соответствующую ей страницу, далее на страницы с информацией о товарных единицах и их зарезервированным местоположением.

### Добавление и удаление товара, чтение и редактирование данных о нем
1) Кнопка "+" используется для создания нового товара и ведет на страницу создания иили изменения товара. После нажатия "сохранить" будет создана новая позиция согласно заполненным данным.
2) Поиск существующих товаров происходит посредством заполнения панели фильтров и нажатия кнопки "поиск".
3) Когда сворачиваемый элемент нужной строки товара развернут, становятся доступны кнопки/ссылки "Изменить"/"Удалить"/"Страница товара". Кнопка "Изменить" ведет на страницу "Создание или изменение товара", которую можно использовать для редактирования информации о товаре, будет изменена существующая позиция. Нажатие кнопки "Удалить" удалает соответствующую позицию.
4) В отдельной странице "Категории товаров" аналогично поиску товаров находятся нужные категории, строки категорий так же разворачиваются на "Изменить"/"Удалить".
5) "Изменить" / "+" изменяет существующую / добавляет новую категорию. "Удалить" - удаляет.

### Добавление и удаление поставщиков и потребителей, чтение и редактирование данных о них
По аналогии с разделом "Товары":
1) Через меню зайти на страницу "Поставщики"/"Потребители".
2) Кнопка "+" используется для создания нового поставщика/потребителя и ведет на страницу создания или изменения поставщика/потребителя.
3) Заполнить контактную информацию и нажать "сохранить".
4) Поиск существующих товаров происходит посредством заполнения панели фильтров на странице "Поставщики"/"Потребители" и нажатия кнопки "поиск".
5) Когда сворачиваемый элемент нужной строки товара развернут, становятся доступны кнопки/ссылки "Изменить"/"Удалить"/"Страница поставщика"/"Страница потребителя". Кнопка "Изменить" ведет на страницу "Создание или изменение поставщика/потребителя", которую можно использовать для редактирования информации о поставщике/потребителе, будет изменена существующая позиция. Нажатие кнопки "Удалить" удалает соответствующую позицию.

# часть 2
## Описание страниц

- Сверху каждой страницы есть меню, состоящее из кнопок, ведущих на страницы Товары, Потребители, Поставщики, Заказы, Поставки, Категории товаров, Главная

### Главная
- Не считая меню: базовая информация о сайте (проекте) и навигации. С помощью меню можно пререйти на другие страницы.

### Товары

- Панель фильтров: текстовые поля для: id, наименование, категория, id категории, единицы измерения (перечислимый тип), габариты (перечислимый тип), срок хранения (временной диапазон), крупногабаритный (флажок). Каждый из них может заполняться для сужения поиска.

- Рядом кнопка "поиск" и "+". Снизу список товаров (строки товаров, результат запроса), соответствующих текущим фильтрам. Кнопка "поиск" обновит список товаров. Кнопка "+" позволяет создать новую позицию товаров.
Список можно пролистывать (если все строки не умещаются в экран)

- Каждая строка в списке товаров - кнопка, разворачиваемая в три кнопки "Изменить"/"Удалить"/"Страница товара". Первая позволяет поменять информацию о товаре, вторая удаляет позицию, третья переход на "Страница товара".

### Страница товара

- Сверху (но ниже меню) информация о товаре (наименование, категория, описание, срок хранения, габариты, единицы измерения). 

- Чуть ниже панель фильтров (id поставщика, местоположение (комната/полка), количество).

- Кнопка "поиск" для обновления списка подходящих имеющихся товарных единиц. Ниже сам список товарных единиц, который может пролистываться. 

- Строка товарной единицы при нажатии разворачиватеся в подробную информацию о товарной единице (с местоположением, количеством, датой поступления, ссылками на соответствующие поставки/заказы, а также флагами "в наличии" и "зарезервировано". Эти флаги получаются из таблиц поставок/заказов))

### Создание или изменение товара

- Текстовые поля для каждого поля данных в базе данных (наименование, описание, габаирты, категория, срок хранения, единицы измерения).

- Кнопки "Сохранить" и "Отмена" (сбрасывает заполненные поля)

### Потребители
- Панель фильтров: текстовые поля для id, наименования, адреса, номера телефона, электронной почты

- Кнопка "Поиск", обновляющая список результатов

- Кнопка (+), переходит на страницу "Создание или изменение потребителя"

- Список всех потребителей (результат запроса, пролистывается)

- Каждая запись является сворачиваемым элементом, который раскрывается в кнопки изменения и удаления, кнопку страницы потребителей.


### Страница потребителя 

-  Отображение подробной информации о потребителе. Также содержит ссылку, ведущую на страницу "Заказы" с этим потребителем, установленным в качестве фильтра

### Создание или изменение потребителя
- Поля  ввода данных (наименование, описание, адрес, номер телефона, электронная почта)
- Кнопки "Сохранить" и "Отмена"
  
### Поставщики

- Панель фильтров: поля ввода текста для id, наименования, адреса, номера телефона, электронной почты

- Кнопка "Поиск", обновляющая список результатов

- Кнопка (+), переходит на страницу "Создание или изменение поставщика"

- Список всех поставщиков (результат запроса, пролистывается)

- Каждая запись является сворачиваемым элементом, который раскрывается в кнопки изменения и удаления, страницы поставщика.

### Страница поставщика 

-  Отображение подробной информации о поставщике. Также содержит ссылку, ведущую на страницу "Поставки" с этим поставщиком, установленным в качестве фильтра


### Создание или изменение поставщика
- Поля  ввода данных (наименование, описание, адрес, номер телефона, электронная почта)
- Кнопки "Сохранить" и "Отмена"
  
### Заказы

- Панель фильтров: Поля поиска для id потребителя, товар (наименование), количество (диапазон), время (диапазон), выполнен (флаг)

- Кнопка "Создать новый" (+), переходит на страницу "Создание или изменение заказа"

- Список всех заказов (результат запроса, пролистывается)

- Каждая запись является сворачиваемым элементом, который раскрывается в кнопки изменения и удаления и страницу заказа.


### Страница заказа

- Отображение подробной информации о заказе. Также есть ссылка на страницу потребителя.
- Имеет кнопку для выполнения заказа (Меняет значение флага, товарные единицы удаляются из базы данных)

### Создание или изменение заказа
- Поля  ввода данных (потребитель  id, товар id, количество, время)
- Кнопки "Сохранить" и "Отмена"
  
### Поставки

- Панель фильтров: Поля поиска для id потребителя, товар (наименование), количество (диапазон), время (диапазон), выполнен (флаг)

- Кнопка "Создать новый" (+), переходит на страницу "Создание или изменение поставки"

- Список всех поставок (результат запроса, пролистывается)

- Каждая запись является сворачиваемым элементом, который раскрывается в кнопки изменения и удаления и страницу поставки. !Удаление незавершенной поставки удалит записи о товарных единицах и зарезервированном под них месте в базе данных.

### Страница поставки
- Отображение подробной информации о поставке. Также есть ссылка на страницу поставщика.
- Имеет кнопку для выполнения поставки (меняет флаги)
  
### Создание или изменение поставки
- Поля  ввода данных (поставщик  id, товар id, количество, время)
- Кнопки "Сохранить" и "Отмена" 

### Категории товаров

- Панель фильтров: Поля поиска (id, наименование)

- Список категорий (записи разворачиваются на кнопки Изменить, Удалить, и кнопку Товары со ссылкой на "Товары" с запросом, ограниченным этой категорией)

- Кнопка (+) и "Изменить" переводят на страницу "Создание или изменение категории товара", кнопка "Удалить" удаляет пустую категорию.

### Создание или изменение категории
- Поле  ввода данных (Наименование)
- Кнопки "Отправить" и "Отмена"
